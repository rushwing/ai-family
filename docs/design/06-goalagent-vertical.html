4. Dogfooding：第一个 Goal 就是本项目

# GoalAgent 录入的第一个目标（M1 上线日执行）
Goal: "ai-family 平台建设"           # 中期目标，周期 = M1–M5
Milestones: M1 基座+切片 / M2 编排治理 / M3 知识学习 / M4 领域补全 / M5 K3s
每周计划: 从 harness/tasks/features/ 的 REQ 状态同步（人工录入起步，M2 后由 goal-mcp 工具读仓库自动核对）
日节拍: 周末档开发任务 + 工作日轻量任务（评审回复/文档修订）
周报: GoalAgent 周报必须含：REQ 状态变化、偏差说明、下周窗口任务批次（喂给 Claude Code Pro 窗口，见 05 §3）
🎯 这一步的真实意义：M1 交付的不只是功能，而是把项目管理职责从人移交给系统——之后任何里程碑逾期，是 GoalAgent 的周报先发现，不是人的愧疚感先发现。
