## 前置条件
- IdP（ADR-008 选型，Keycloak）已起；realm + client(code+PKCE) 已配
- 4 个家庭成员账号；角色映射 admin / adult / kid

## 步骤
1. 拉取 `.well-known/openid-configuration`（OIDC discovery）
2. 各成员走 code+PKCE 换取 ID/Access token
3. 解析 JWT，校验 `sub`、角色 claim、`aud`、签名（JWKS）
4. 用过期/篡改 token 调受保护接口
